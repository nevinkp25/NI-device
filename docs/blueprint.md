# **App Name**: SwiftBite

## Core Features:

- Staff ID Input: Input field for staff ID number with a loading state on submit.
- Menu Display: Display food menu in a two-column layout with category tabs. Category tabs will dynamically update based on the section in view during scroll.
- Add to Cart: Add items to the cart with a + icon. Adding an item reveals the quantity button.
- Floating Cart: Display a floating basket icon with a button to proceed to checkout.
- Checkout Display: Display items in the cart with subtotal and total amounts. Option to choose Cash or Card payment methods.
- Payment Processing: Process payment based on the selected method (Cash or Card).
- Success Animation: Display an animation of a receipt revealing and a success icon after successful payment, with a button to return to the home screen.

## Style Guidelines:

- Primary color: Vibrant Coral (#FF7F50) to create an energetic and modern feel.
- Background color: Light beige (#F5F5DC) for a clean and appetizing backdrop.
- Accent color: Teal (#008080) for call to actions and highlights to stand out.
- Font pairing: 'Poppins' (sans-serif) for headlines and 'PT Sans' (sans-serif) for body text, providing a modern and readable combination.
- Use minimalistic and clear icons for food categories and actions (add, remove, checkout).
- Responsive layout optimized for iPhone mobile size, with clear separation of sections and easy navigation.
- Smooth transitions and subtle animations on button clicks, adding to cart, and payment success for enhanced user experience.
